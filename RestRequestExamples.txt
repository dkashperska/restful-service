1. Create circle
POST
/restfull-service/rest/circle/create
Content-Type=application/json
JSON={
  "radius": 3
}
------------------------------------------
2. Get circle
GET
/restfull-service/rest/circle/4
------------------------------------------
3. Update circle
PUT
/restfull-service/rest/circle/update
Content-Type=application/json
JSON={
  "id":4,
  "radius": 7
}
------------------------------------------
4. Delete circle
DELETE
/restfull-service/rest/circle/4
------------------------------------------
